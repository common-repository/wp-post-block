WP Post Block is an easy way to insert your posts into your content without a single line of code. After you install WP Post Block, you will have new blocks on Gutenberg, aka Block Editor, ready to use. You can search your posts by title and category, to choose any post you want to display anywhere in your content.

== Description ==

Chegou um plugin com blocos para Posts. Com o WP Post Block é possível destacar artigos numa página ou num outro artigo, simplesmente pesquisando e escolhendo qual o artigo que se pretende incluir. Automaticamente o artigo aparece no conteúdo, com link direto para o detalhe do post. É ainda possível escolher os campos a apresentar em cada um dos artigos, e fazer algumas costumizações. Desta forma, é possível criar páginas completamente diferentes só com diferentes combinações dos mesmos blocos. É ainda disponibilizado um bloco secção que permite agrupar os restante blocos dentro dum container com um título associado.

> With WP Post Block you can enrich your website inserting and customizing from one to multiple posts.

Com o WP Post Block consegues alterar um post muito rapidamente. Este plugin foi contruído para permitir flexibilidade mas também muita facilidade e rapidez na atualização de conteúdos.

=== Blocks ===

* Post headline: mostra o artigo num formato de destaque (foto maior), com link para o seu detalhe e com os campos que o utilizador selecionou na configuração do bloco.
* Post card: mais adequado para listagens ou criação de blocos de relacionados num post, mostra num cartão o artigo escolhido com link para o seu detalhe e com os campos que o utilizador selecionou na configuração do bloco.
* Posts container: cria um área onde é possível incluir outros blocos, e permite dar um título a esta área.

=== Features ===

POST HEADLINE

* Incluir um bloco de destaque na sua página para destacar um post como manchete / destaque principal duma página. A altura deste bloco adota no máximo a altura do ecrã.
* Apresentar o autor do bloco de destaque
* Apresentar o excerto do bloco de destaque
* Aplicar um filtro de contraste ao texto do bloco de destaque

POST CARD

* Incluir um bloco de artigo na sua página para apresentar um post em listagem.
* Apresentar a categoria do bloco de artigo
* Apresentar o autor do bloco de artigo
* Formato por omissão: Imagem de destaque com informação do artigo abaixo da mesma
* Formato opcional: Imagem de destaque com a informação do artigo à direita da mesma

POSTS CONTAINER

* incluir um bloco grelha na sua página para agrupar manualmente posts.
* alterar o título do bloco grelha
* aplicar uma cor de fundo ao bloco de artigo

SEARCH

* Pesquisar qual o post que vai incluir no bloco
* Filtrar, na pesquisa, os posts por título e/ou por categoria
* Visualizar, nos resultados da pesquisa, o título e imagem do post de forma a facilitar a escolha
* Nos resultados de pesquisa, clicar num post e automaticamente este aparece no bloco que tinha sido adicionado/selecionado
* Arrastar qualquer resultado de pesquisa de posts, apresentados na interface de configuração do bloco, para qualquer outro bloco de artigo existente na página, alterando assim o artigo associado a esse mesmo bloco.


=== Vantagens em utilizar o WP Post Block ===

* Criar homepages mais atrativas com os seus conteúdos
* Incluir numa homepage já existente blocos com artigos (sem ser uma listagem fechada que impossibilidade de escolher a ordem)
* Criar páginas específicas de um tema
* Enriquecer os seus posts com relacionados, ou outros conteúdos de interesse
* Enriquecer os sites dos seus clientes com mais blocos Gutenberg que aumenta a potencialidade do produto

=== Simple ===

The use of this plugin requires no knowledge in programming and no code writing is necessary. Simply install it, insert post into your content and the plugin will work correctly.

=== Compatibilidade com temas ===

Este plugin foi testado com os temas free mais populas da comunidade WordPress. O WP Post Block funcionará com os temas que são compatíveis com o Gutenberg.

=== Compatibilidade com Page Builders e outros blocos ===

Este plugin adiciona blocos novos ao Gutenberg, podendo ser utilizado com outros plugins que também providenciem blocos ou então com page builders desde que compatíveis com o Gutenberg

=== Mobile responsive ===

This plugin is optimized to work properly on mobile devices.

=== Performance ===

Este plugin foi construido para lidar com milhares de artigos. Não precisas de te preocupar.

== Installation ==

1. Upload `wp-post-block` to the WordPress `plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert any brand new `Post Blocks` into your content and customize
